\subsection{Detailed Use Case Diagrams}

The following subsections present a detailed use case diagram per functional subsystem, drawn using UML notation. Actors appear outside the system boundary rectangle. Ellipses represent use cases. Solid lines denote associations; dashed lines with open arrowheads denote \texttt{<<include>>} or \texttt{<<extend>>} relationships.
