\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
Anhoc is a web-based mathematics learning platform designed to provide structured theory content, automatically generated practice exercises, grade-level tests, and progression tracking for students. The system also includes administration capabilities for lesson management, question template authoring, role-based access control, and achievement configuration. This thesis documents the design and implementation of Anhoc based on the current codebase in the repository.

The project follows a client-server architecture in which a Next.js frontend communicates with a Node.js and Express backend through REST endpoints. Prisma is used as the data access layer, while PostgreSQL stores application data such as users, lessons, question templates, attempts, mastery records, achievements, and audit evidence. A notable implementation decision is the use of persisted question snapshots, which allows each generated exercise to remain stable after creation and supports reproducible scoring, review, and reporting.

This thesis presents the problem context, functional and non-functional requirements, system architecture, data model, implementation details, and an evaluation of the current solution. It also discusses limitations and future improvements needed to move the platform from a capable academic software project to a more mature production learning system.
